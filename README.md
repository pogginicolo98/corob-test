# Corob test

Twitter-like web application based on Django and React. Test for technical interview with Corob

## Run Locally

Clone the project

```bash
  git clone https://github.com/pogginicolo98/corob-test.git
```

Go to the project directory

```bash
  cd corob-test
```

Build docker images

```bash
  docker compose build
```

Start docker

```bash
  docker compose up
```
